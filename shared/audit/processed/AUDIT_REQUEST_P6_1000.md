## 三、Conductor 分析 — 三种可能解读

### 假说 A: 无 GELU 确实有害
- 纯线性投影 = rank-2048 矩阵分解, 无法建模 DINOv3 特征空间中的非线性边界
- P6 car_P=0.054 远低于 Plan L=0.140, 且同期 bg_FA 也更好 (0.323 vs 0.407)
- 如果 bg_FA 更低但 car_P 也更低, 说明模型在做更保守的预测 (少预测 → 少 FP 但也少 TP)
- **支持证据**: P6 off_th=0.250 比 Plan L=0.242 更差, 无 GELU 对方向回归也无帮助
- **如果成立**: 需要恢复 GELU (推翻 BUG-30 结论), 或用其他非线性 (ReLU, SiLU)

### 假说 B: 仅是类振荡时序差异 + LR mult 干扰
- P6 proj LR 2x → 类别激活更快 → 在 @1000 时已进入类振荡阶段
- Plan L proj LR 1x → 类别激活更慢 → 在 @1000 时 car 还在独占阶段
- P6 @500 car_P=0.073 接近 P5b@500=0.080, 说明架构基本正常
- **如果成立**: P6 @1500/@2000 car_P 可能回升 (类振荡周期 ~1000 iter)
- **支持证据**: P6 @500 bg_FA=0.163 极优, 说明无 GELU 对背景判别有巨大优势

### 假说 C: gt_cnt 差异导致不公平对比
- P6 val 可能使用了不同的标注配置 (truck GT +95% 极度异常)
- 如果 val set 确实不同, 所有跨实验对比无效
- **如果成立**: 需要先修正 val 配置, 重新评估
