- car_R = 0.8195 (vs @2000: 0.8124) — 稳定高位 ✅
- car_P = 0.0451 (vs @2000: 0.0571) — 下降 21% ⚠️
- bg_FA = 0.3240 (vs @2000: 0.2073) — **上升 56%, 超红线 0.25** ❌

   - 分析 per-class FP 贡献: 是 car 的 FP 还是其他类别的 FP？
   - 对比 ORCH_024 @4000 的 FP 模式

3. **训练趋势判断**: bg_FA 是否还在恶化？检查 loss 组分是否有 cls_loss 下降、reg_loss 上升等不平衡迹象

4. **可行修复方案评估**:
   - 置信度阈值后处理能否改善 car_P 和 bg_FA？
   - loss 加权 (边缘 cell 降权) 是否可行？
