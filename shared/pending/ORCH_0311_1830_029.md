CEO 指出 overlap 模式过于激进: 大量边缘 cell 仅微小重叠却被标为前景。
经 CEO 指导完成两阶段过滤设计并已合入 GiT master (commit `a64a226`):
- Stage 1: `vis >= 10%` (object-level, 拒绝超出画面的物体)
- Stage 2: `IoF >= 30% OR IoB >= 20%` (cell-level, 过滤边缘噪声 + 保护小目标)
- IoB 分母用 full bbox 面积 (未裁剪)

20 样本验证: FG cell 减少 30.3%, 对象保留率 96.9%, 0 对象因过滤丢失。
