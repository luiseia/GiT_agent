#### 2. @14000 val 自动完成后, 记录默认结果 (score_thr=0 baseline)
从日志中提取 @14000 val 结果。这是 score_thr=0 的 baseline。

#### 3. 暂停训练
```bash
# 获取训练主进程 PID
TRAIN_PID=$(ps aux | grep "tools/train.py" | grep "local_rank=0" | grep -v grep | awk '{print $2}')
# 发送 SIGSTOP 暂停 (不终止)
kill -STOP $TRAIN_PID
# 等待 GPU 释放
sleep 30
nvidia-smi --query-gpu=index,memory.used --format=csv,noheader
```

**注意**: 如果 SIGSTOP 不释放 GPU 内存, 则需要 kill 训练后从 iter_14000.pth resume。

#### 4. 串行执行 score_thr 消融
```bash
cd /home/UNT/yz0370/projects/GiT
conda activate GiT

# 使用 4-GPU DDP (每次 ~56min) 而非 1-GPU (每次 ~3.5h)
for thr in 0.1 0.2 0.3 0.5; do
  echo "=== score_thr=$thr === $(date)"
  python -m torch.distributed.launch --nproc_per_node=4 --master_port=29501 \
    tools/test.py configs/GiT/plan_full_nuscenes_multilayer.py \
    /mnt/SSD/GiT_Yihao/Train/Train_20260312/full_nuscenes_multilayer_v4/iter_14000.pth \
    --launcher pytorch \
    --cfg-options val_evaluator.score_thr=$thr
  echo "=== Done score_thr=$thr === $(date)"
done
```

**预计总耗时**: 4 × ~56min ≈ **~4h**

#### 5. 恢复训练
```bash
# 如果用了 SIGSTOP:
kill -CONT $TRAIN_PID

# 如果 kill 了训练:
python -m torch.distributed.launch --nproc_per_node=4 --master_port=29500 \
  tools/train.py configs/GiT/plan_full_nuscenes_multilayer.py \
  --resume /mnt/SSD/GiT_Yihao/Train/Train_20260312/full_nuscenes_multilayer_v4/iter_14000.pth \
  --launcher pytorch
```
