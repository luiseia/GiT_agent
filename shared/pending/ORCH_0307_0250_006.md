## 任务: DINOv3 离线特征预提取

### 方案描述
对 nuScenes-mini 全部 323 张图片, 用 DINOv3 跑一次前向传播, 保存中间层特征 (Layer 16-20) 为 `.pt` 文件. 训练时直接加载 `.pt` 替代 Conv2d PatchEmbed, 不占额外显存.

### 实现步骤

1. **编写预提取脚本** (`scripts/extract_dinov3_features.py` 或类似):
   - 加载 DINOv3 ViT-7B 模型 (FP16)
   - 遍历 nuScenes-mini 323 张图片
   - 对每张图, 前向传播到 Layer 16-20
   - 保存各层输出: `{image_id}_layer{N}.pt`, shape `(N_patches, 4096)`
   - 保存路径: `/mnt/SSD/GiT_Yihao/dinov3_features/`

2. **修改 GiT 数据加载**:
   - 在 `vit_git.py` 中添加加载 `.pt` 的路径
   - 替代 `dinov3_patch_embed` 的 Conv2d 输出
   - 添加 `Linear(4096, 768)` 投影层 (可训练)
   - 用 config 参数控制: `use_preextracted_features=True`

3. **GPU 使用**:
   - 预提取可用单卡 (DINOv3 7B FP16 约 14GB, A6000 48GB 充足)
   - 只用 GPU 0 或 2
   - 预提取完成后释放 GPU
